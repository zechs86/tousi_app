## 開発の進め方（重要・ハマりどころ）
- コード変更 → `git push origin main` で Streamlit が自動再デプロイ。
- push前に `gh auth login`（トークンが切れがち）。workflow(.github/)変更時は `gh auth refresh -s workflow` で workflow スコープ追加。
- gh: `C:\Program Files\GitHub CLI\gh.exe`。デバイスログインはバックグラウンド実行で8桁コードを読み取る方式が確実。
- GitHub Secret登録は PowerShell では `gh secret set NAME --body $val`（パイプはstdin文字化け）。
- ntfy: Titleヘッダは英数字のみ。本文~4096バイト上限。トピックは strip 必須。
- AIモデルは `config.AI_MODEL`(既定 claude-sonnet-4-6)。1回約¥4。最高精度は claude-opus-4-8。
- 当日未確定バー対策: 株価dfは `dropna(subset=['Close'])`。
