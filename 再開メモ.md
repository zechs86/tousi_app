- 📊 銘柄分析 … チャート＋ファンダ。上部に **⭐お気に入り**（ワンタップ表示）＋☆登録/⭐解除（`favorites.py`）
- 🤖 AI分析 … Claudeが1銘柄分析(業績/機関投資家/目標株価/なぜ今か)。`ai_analysis.py`。手動1回約¥4
- 💬 AI相談 … **既定オフ**(`config.AI_CHAT_ENABLED=False`・従量課金のため)。コードは`ai_chat.py`に有り
- 💰 ペーパー … 仮想売買・資産推移グラフ・勝率・利確🎯/損切り🛑(購入後も確認/変更可)。`paper.py`

裏方: `risk.py`(暴落検知) / `notify_job.py`(朝夜通知=買い+⚠️リスク+🤖AIコメント+📱アプリURL) /

## データ永続化（DB）= 設定済み（重要）
- `src/store.py` が **Upstash Redis(無料・REST)** に保存。未設定ならdata/kv/にファイル。
- 認証: `UPSTASH_REDIS_REST_URL` / `UPSTASH_REDIS_REST_TOKEN`。
  - 手元: `src/secret_local.py` に記入済み。
  - クラウド: **Streamlit Secrets に登録済**（ユーザーが貼付・Reboot済）。
  - GitHub Actions: **Secrets登録済**（`gh secret set --body`）。
- キー: ペーパー=`paper:<user>` / お気に入り=`fav:<user>`。→ 人ごと・クラウドでも消えない。
- Upstash URL例: https://intense-bat-136130.upstash.io （アカウントはGitHub:zechs86でログイン）

## 開発フロー / ハマりどころ
- 変更→`git push origin main`でStreamlit自動再デプロイ。GitHub: **public** `zechs86/tousi_app`。
- push前に `gh auth login`(トークン切れがち)。workflow変更時は `gh auth refresh -s workflow`。
- gh: `C:\Program Files\GitHub CLI\gh.exe`。デバイスログインはバックグラウンド実行で8桁コードを読む方式。
