# 投資ダッシュボード（tousi_app）— つかい方

最終更新: 2026-06-26

日本株を「テクニカル × ファンダ × ニュース」で分析し、**買いどき（今ここ！）をスマホ通知**で教えてくれるアプリです。
クラウド（Streamlit Cloud）で常時稼働しており、**PCを起動しなくてもスマホからどこでも**使えます。
※投資助言ではなく判断材料です。最終判断はご自身で。

---

## 1. スマホで使う（メイン）

- **アプリURL**: https://tousiapp-halx5hjmpkl8fkzcyq2gqn.streamlit.app
- **朝夜の自動通知**（ntfy・無料）: 平日 8:15 / 18:30(JST) に、その日の状況がスマホに届きます。
  - 内容: 🟢買い候補 ／ ⚠️急変リスク ／ 🤖AIの一言 ／ 🎯利確・🛑損切りの到達＆接近 ／
    🟢お気に入りの押し目 ／ 🎁イオン優待カウントダウン ／ 📅決算が近い銘柄 ／ 📱アプリを開くリンク

### サイドバーの「👤利用者」
名前を入れると、ペーパー口座・お気に入り・優待設定が**あなた専用**に分かれます（友人と共有OK）。空欄=ゲスト。

## 3. ⚙️ 設定でできる調整（コード不要）

「⚙️ 設定」ページで保存すると、アプリと**次回の自動通知にも反映**されます。

- **押し目とみなすRSI**（既定40・小さいほど厳しめ）
- **押し目買いアラート** オン/オフ
- **優待リマインド開始日数**（既定75）／**決算リマインド日数**（既定14）
- **利確/損切り 接近予告(%)**（既定2.0・0でオフ）
- **通知する上位件数**（既定8）
- **通知のAIコメント**（月¥20〜40の課金）／**ニュースAI要約ボタン**（押した時だけ¥1〜2）

> 💡 イオン優待を安く仕込むコツ: 🗓️予定で**優待の年間価値**（年間イオン利用額×3%が目安）を入れると
> 「総合利回り◯%」が出ます。お気に入りに 8267.T を入れておけば、**下げ止まって押し目に転じた瞬間に通知**が来ます。

---

## 4. お金（課金）について

- 株価・ニュース・通知は**すべて無料**（yfinance / GoogleニュースRSS / ntfy）。
- 課金が出るのは **AI機能だけ**:
  - 🤖AI分析（手動・1回約¥4）／ 通知のAIコメント（既定オン・月¥20〜40）/ ニュースAI要約（既定オフ・1回¥1〜2）/ 💬AI相談（既定オフ）。
  - **気になる場合は ⚙️設定 でAI系をオフ**にすれば一切かかりません。

## 5. 開発者向けメモ

- 構成: `src/` にモジュール、ルートに `notify_job.py`（GitHub Actionsの朝夜通知）。
- 更新の反映: `git push origin main` → Streamlit Cloud が自動再デプロイ。
- 設定の保存先: ペーパー/お気に入り/優待値/⚙️設定は **Upstash Redis**（無料REST）に永続化。未設定時は `data/kv/` にファイル。
- 鍵類は `src/secret_local.py`（gitignore）／クラウドは Streamlit Secrets／通知ジョブは GitHub Secrets。
- 日本語パスのSSL対策を `src/_net.py` で実施。
- 詳しい現状は `再開メモ.md`（毎セッション更新）を参照。
