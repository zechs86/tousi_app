# 投資分析アプリ（tousi_app）— 状況とつかい方

最終更新: 2026-06-24

## このアプリの目的
日本株・米国株をテクニカル＆ファンダで分析し、「買いどき／売りどき（今ここ！）」を
スマホ通知で教えてくれるアプリ。最終的には短期の自動売買も視野。

## 方針（決定ずみ）
- 長期の土台 → 楽天証券・NISAでインデックス積立（オルカン or S&P500）。ほったらかし。
- 短期トレード → 三菱UFJ eスマート証券・特定口座（口座開設申請中）。将来 kabuステーションAPIで自動売買。
- 通知 → ntfy（無料）。トピック(合言葉)は src/secret_local.py に保存（公開しない）。設定済み・テスト成功。
- 使う時間 → 平日 朝8:30〜9:30 / 夜18:30〜20:00。→ 朝夜2回の通知が目標。
- 資金/リスク → 10万円スタート・大きく狙う・手動の頻繁売買は難しい→通知＆自動化重視。スイング中心。

## 大事な学び（バックテストで判明）
- 単純なテクニカル戦略は、たいてい「ずっと持つ(buy&hold)」に勝てない。
- 「暴落を無条件に買う(落ちるナイフ)」は平均で負ける。
- timing戦略の価値は「儲けを増やす」より「大ケガを避ける」こと。
- → 一発逆転は狙わず、損切りを守り退場せず、複利で育てる。
